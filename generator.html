Generate a modifiers dictionary.

Usage: Use this to generate a dictionary of modifiers (Ctrl, Shift, Alt, Super (Win), Function keys, and all combos/shortcuts.
Choices:Select "Phoenix" to use Phoenix letters instead of Plover.
Execute!:Click "Generate Dictionaries" Result:You will get three single stroke dictionaries, and three two stroke dictionaries:

  • Real: When loaded in Plover this will execute the modifiers when stroked.
  • Practice: When loaded in Plover (use _instead of_ Real) print out the modifiers when stroked for practice purposes.
  • StenoJig: This is in StenoJig Translations format. Use this along with Practice dictionary loaded in Plover to practice modifiers using Stenojig exercises.
  • StenoJig All Exercises: This is a StenoJig exercise (using Practice output) for ALL generated single stroke modifier entries.
